# Student_performance_analysis

Week 1 Internship Project - Student Performance Dataset Analysis using Pandas, NumPy, Matplotlib, and Seaborn.

## Student Performance Dataset Analysis

# Objective

The primary goal of this project is to analyze student performance data and uncover relationships between academic outcomes and factors such as study habits, attendance, and demographic characteristics.

# Tools Used

* Python
  
* Pandas

* Matplotlib
  
* Seaborn

# Key Insights

* Students with more study hours tend to score higher.
  
* Attendance positively affects final grades.
  
* Gender distribution is balanced.
  
* Final grades are concentrated in a specific range.
  
* Strong correlation exists between study hours and grades.

# Conclusion

The analysis indicates that study time and attendance are significant contributors to student success. Encouraging consistent study routines and regular class participation can help improve overall academic performance.
